# markdown编辑器

![预览图1](image/a.png)

#### 介绍
轻量级的markdown编辑器

#### 软件架构
界面简洁，基于markdown.js


#### 安装教程

1.  下载发行版
2.  解压到网站服务器/虚拟主机
3.  访问`index.html`，开始使用

#### 使用说明

1.  在输入框里输入markdown
2.  灰色区域是markdown对应的html代码
3.  粉色区域是效果预览
